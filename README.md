# data-visualization-repo
Data analysis and visualization projects created during my studies

1. Analysis of Amazon.com sales data
  * Performed exploratory data analysis on Amazon Sales dataset (https://www.kaggle.com/datasets/anandshaw2001/amazon-sales-dataset), which was posted on Kaggle.com by Anand Shaw, using Tableau business intelligence tool.

2. Analysis of coffee-based products sales data
   * Performed exploratory data analysis on Coffee Shop Sales dataset, posted on Kaggle.com, using QlikView business intelligence tool.

3. Predictive sales analysis in coffee vending machines using machine learning algorithms
   * Performed exploratory data analysis using NumPy, Pandas, Matplotlib and Seaborn libraries, as well l as carried out predictive sales analysis of coffee vending machines using Random Forest Regressor algorithm
